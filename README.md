## react hooks+ React Native 抖抖APP (仿抖音)
### 1.体验
ios: 真机or模拟器

### 2.运行
```shell
git clone
yarn
$ yarn ios

```
### 3.技术栈
- react 16.13
- react-native 0.63
- react-native 核心API： Animated， PanResponder
- redux
- redux-immutable
- react-navigation
- react-native-vector-icons
- react-native-video
- react-native-camera
- react-transition-group

### 4.主要页面

- 1.视频播放页
> 视频流 滑动切换视频
> 2.关注页
> 关注列表
- 3.拍摄视频页
> 视频拍摄 查看 放大缩小 前后摄像头切换
- 3.消息页
> 消息列表
- 4.我的页
> 个人页面 资料 视频列表
